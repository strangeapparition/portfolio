A couple years ago, my friend Patrick told me about this idea he had to create an app that would automatically generate a party playlist based on who is in attendance. We had a preliminary chat about the idea and potential scope, and I agreed to join. I’d wanted to learn more about design, so I took this as a great opportunity to learn in my spare time and tackle a new and interesting challenge.

We’ve gone through a couple iterations, each time having a party with friends for some convenient user testing. The first version was alright, but the interface needed work. The second version crashed at the party! Issues were piling up as a result of last minute changes and scope inflation, so the third version was built from the bottom up.

These are some of the designs for the first iteration. The core idea remained but the interface needed work.

For version three we needed to be more organized so we signed up for Trello. It was then very easy to keep track of tasks and bugs, and share design files. With this we were able to watch the project grow.

I started with some sketches and notes, which we talked over to discuss the flow of the app and what features to include or hold off on.

I like having single sheets of paper to flip around so I bought some cheap graph paper and one of those tacky clipboards with storage to keep the extra paper, pencil, and markers in. A little dorky, but it works for me!

I worked in sections, so after sketching a few pages, I drew them up in Adobe Illustrator. I did this so we could work simultaneously, with me one section ahead sketching and designing.

The designs were done in Adobe Illustrator. I wanted the design to be flat but a bit dynamic, so I used different colors from the logo in each section of the app.

The app was designed mobile-first, with bigger screen sizes in mind. More screen sizes will be supported in the future, along with other features not included in this release.

Give Jukebot a try!

To start a party you’ll need a Jukebot account, a premium Rhapsody account, and a browser that can run Flash. Once signed in you can create a party.

To join a party you’ll need a phone (recommended) or any device with a web browser, a Jukebot account, and the party name and password from the host.
