Jukebot is a semi-automated party playlist generator. It's made to let everyone at a party contribute to the music selection.

I've been a front end developer the past few years and decided I wanted to expand my skill set into design. This project seemed like the perfect way to do that! And so I became the sole designer for the project.

We started with talking through the idea. How could this app work on a small scale - like house parties? And how could it work on a larger scale - like bars? The scope of the project, we decided, needed to be small for now.

Our initial concept isn't far off from what we have today.

These wireframes are from the very first version of Jukebot. The core idea remains but the interface is vastly improved.

This time around we decided to redo the application from the ground up. The app is built with ReactJS and uses the Rhapsody API.
